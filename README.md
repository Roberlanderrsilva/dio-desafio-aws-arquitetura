# Gerenciamento de Instâncias EC2 na AWS - Desafio DIO
Este repositório contém o diagrama de arquitetura de uma solução AWS, criada para o desafio prático da DIO.
## Arquitetura do Projeto
O diagrama representa um fluxo de processamento de dados:
 1. Amazon S3: Armazenamento de objetos e gatilho do fluxo.
 2. AWS Lambda: Função para processamento automático de arquivos.
 3. Amazon EC2: Servidor para gerenciar a aplicação final.
## Ferramentas Utilizadas
 * Draw.io: Criação do diagrama.
 * Ícones oficiais AWS: Representação técnica.
## Aprendizados
Consolidei conhecimentos sobre documentação de infraestrutura em nuvem e a integração entre serviços serverless (Lambda) e instâncias de computação (EC2).
